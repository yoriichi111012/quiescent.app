Core launcher work

Built around the chores players repeat.

Quiescent was built with focus on practical things: setup, launching, modding, instances, account handling, diagnostics, and updates.

Version and launch management

Choose official releases, snapshots, older versions, locally installed builds, or Fabric versions. Quiescent can prepare game files automatically and show progress while downloads run.

Separated instances

Create isolated setups with their own versions, mods, saves, configs, resource packs, shader packs, folders, and icons, then open or delete them from the launcher.

Mods without folder hunting

Search installed mods, inspect names, versions, IDs, and icons, enable or disable mods, delete mods, open mods folders, or drag in mod files.

The setup path

One clean path from install to launch.

Quiescent's key target is efficiency: fewer decisions scattered across folders, websites, account dialogs, and update pages.

Account switching without rebuilding the setup

Quiescent supports Microsoft/Mojang accounts, Ely.by accounts, and offline player sessions so the launcher can adapt to the player's actual setup.

  • Add, edit, delete, select, and re-authenticate accounts from the launcher.
  • Support skin compatibility through Microsoft/Mojang and Ely.by/authlib integration.
  • Keep offline sessions available when local files allow launching without internet access.

Certificate installation

Install the certificate before the MSIX package.

The current preview package is signed with a certificate that Windows needs to trust first. Without this step, the MSIX installer may be blocked.

Download both files

Download the certificate and the MSIX package matching your processor architecture from the release.

Open the certificate

Open the certificate file and click Install Certificate.

Use Local Machine

Select Local Machine and click Next.

Choose the store

Choose Place all certificates in the following store, then click Browse.

Trust the certificate

Select Trusted Root Certification Authorities and finish the certificate installation.

Install Quiescent

After the certificate is installed, open the MSIX package and install the launcher.
